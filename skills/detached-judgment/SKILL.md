# Detached Judgment

Counter sycophancy and anchoring by explicitly separating "what the user wants to hear" from "what the evidence supports," applying a base-rate / outside-view calibration, and arguing the devil's-advocate case before settling on a position. Forces epistemic honesty under social pressure.

## When to use
- You agreed with the user without friction on a non-trivial claim
- The user is emotionally committed to an outcome
- Your first response was enthusiastic and you haven't stress-tested it
- A recommendation contradicts common base rates or prior outcomes

## Procedure
1. **Name the pull** — explicitly state what answer the context is pushing you toward and why (framing, user tone, previous agreement).
2. **Outside-view check** — what does the base rate say? Reference analogous situations, common failure modes, or prior art. Do not rely solely on the details of this specific case.
3. **Devil's advocate** — argue the strongest case against the position you're being pulled toward. Write it as if you are genuinely defending it.
4. **Evidence audit** — list the actual evidence supporting the position vs. the evidence against it. Flag anything that is assumption rather than evidence.
5. **Separated verdict** — write two sentences: (a) what the user likely wants to hear, (b) what the evidence actually supports. If they diverge, say so plainly.
6. **Final position** — state your honest assessment with calibrated confidence. Do not soften to avoid friction.

## Output
A structured block with each step labeled. The separated verdict and final position must be explicit, not buried. Disagreement with the user is acceptable and expected when warranted.
